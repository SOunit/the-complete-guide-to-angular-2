# steps of running angular app

- CLI insert js script to `index.html`
- serve `index.html` file
- run js, `main.js`
- `main.js` call module
- replace `app-route` tag with `app-component`

# file flow

- CLI insert js to index.html
- index.html
- main.js
- app.module.ts
- app.component.ts
- render html
